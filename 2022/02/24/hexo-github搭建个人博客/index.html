hexo+github搭建个人博客

2022-02-24
Word count: 835   |   Reading time≈ 3 min
一.搭建博客需要的工具

1、node.js、git、hexo
2、下载hexo npm install -g hexo-cli

二.搭建仓路

1、登录注册github,利用github仓库作为服务器
2、新建仓库

3、 1)仓库名为注册名 (知道注册名和注册邮箱,后面步骤需要用到)

2)注册名.github.io

三.生成SSH Keys

1 创建一个关于博客内容的文件夹,右键打开Git Bash Here,生成ssh ssh-keygen -t rsa -C “注册右键地址”
安装成功以后在本地新建了一个文件夹 路径 C:\Users\Administrator.ssh,打开id_rsa.pub 文件

2、全复制ssh秘钥

3、打开github,右键右上角登录名，找到设置(Settings)点击进入

4、找到SSH and GPG keys,点击确定

新建 ssh key

5、Title 按喜好填写内容,key的内容复制id_rsa.pub的内容,然后Add SSH key

6、成功后页面刷新为

7、测试ssh是否绑定成功：ssh -T git@github.com

四. 本地生成博客内容

1、新建文件夹,(我新建文件夹E盘Blog),进入文件夹，右键Git Bash Here命令行工具,输入hexo init 初始化hexo,操作成功以后Git远程hexo代码会复制到本地
2、输入hexo s，静态生成hexo页面,成功状态,出现本地端口号

3、在页面中显示

五. 发布博客到互联网

1、打开更新到本地的文件夹中的_config.yml(文档或编辑器打开),最下面deploy
deploy:
type:git
repository:博客代码地址
branch:main (修改分支master为main)

完成以后关闭文件

2、博客文件里面右键打开命令行(Git Bash Here),
安装hexo-deployer-git自动部署发布工具: npm install hexo-deployer-git –save

3、成功以后输入 hexo g(hexo generate,生成静态文件) 回车
4、输入 hexo d(hexo deploy,部署网站)
5、成功后首先输入用户名

其次弹出输入密码弹窗,     (注意:此时不用输密码,需要输入令牌)

6、右上角登录下拉列表找到Settings/Developer Settings里面的Personal access tokens,新建一个令牌

Note 名字 Blog(按自己喜好)
Expiration 有效时间30天
Select scopes 为令牌赋权限 全选

生成令牌

复制令牌内容粘贴到输入密码弹出框，用令牌代替密码

7、仓库线上地址为git仓库名字

六.更换Theme

1、网上下载主题，放到Blog文件夹下Themes文件内
2、_config.yml 大概100行 修改theme为下载主题名字

七.引用图片

1、通过将 config.yml 文件中的 post_asset_folder 选项设为 true 来打开

2、当资源文件管理功能打开后，Hexo将会在你每一次通过 命令创建新文章时自动创建一个文件夹。这个资源文件夹将会有与这个文章文件一样的名字。将所有与你的文章有关的资源放在这个关联文件夹中之后，你可以通过相对路径来引用它们，这样你就得到了一个更简单而且方便得多的工作流
3、此时自动创建的文件夹与 title 的名字相同，在文件夹中新建一个images的文件夹,将图片放在images文件夹中,在文章中引用时的格式为
